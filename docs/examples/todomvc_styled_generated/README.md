# TodoMVC Styled Generated References

This directory stores the visual references for the `todo_mvc_physical` native
preview work. The PNG files are the target renders for each theme/mode
combination. The `html/` directory contains the standalone HTML/CSS mockups
copied from:

`/home/martinkavik/Downloads/todomvc_html_themes/todomvc_html_themes/`

The downloaded HTML set contains nine mockups. `classic-light.html` was not
present in that source set, so `classic_light.png` is copied from the local
screenshot:

`/home/martinkavik/Pictures/Screenshots/Screenshot_2026-06-03_21-52-41.png`

The native preview keeps its theme/mode switcher visible, but the switcher is
positioned below the TodoMVC reference content. Visual parity checks should crop
to the TodoMVC content region and exclude the switcher.
